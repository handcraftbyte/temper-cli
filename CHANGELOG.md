# Changelog

All notable changes to temper-cli will be documented in this file.

## [0.2.0] - 2025-01-15

### Added
- `clone` command - Copy public snippets to local library
- Obsidian plugin with code blocks (`temper-run`, `temper-embed`) and commands

### Changed
- `search` now searches public gallery only (cloud)
- `list` now shows local snippets only
- `edit` requires local snippet (use `clone` first for public snippets)
- Removed `--local` and `--remote` flags (no longer needed)


## [0.1.0] - 2025-01-14

- Initial release
